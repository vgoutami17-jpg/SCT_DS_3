# SCT_DS_3

## SkillCraft Technology – Data Science Internship

### Task 3: Decision Tree Classifier

### Objective

Build a Decision Tree Classifier to predict whether a customer will subscribe to a term deposit using demographic and behavioral data from the Bank Marketing Dataset.

### Dataset

Bank Marketing Dataset (UCI Machine Learning Repository)

### Tools & Technologies

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib
* Seaborn
* Google Colab

### Project Workflow

* Imported the dataset
* Explored and understood the data
* Encoded categorical variables
* Split the dataset into training and testing sets
* Built a Decision Tree Classifier
* Evaluated the model using:

  * Accuracy Score
  * Confusion Matrix
  * Classification Report
* Visualized the Decision Tree
* Analyzed Feature Importance

### Results

* Model Accuracy: **87%**
* Successfully predicted whether a customer would subscribe to a term deposit.

### Learning Outcomes

* Data preprocessing
* Feature encoding
* Decision Tree Classification
* Model evaluation
* Data visualization
